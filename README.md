# 🍞 Draw Toast — 머릿속 토스트 꺼내보기

> "같은 단어를 써도 머릿속 그림은 다 다르다. 그걸 눈으로 보자."
> Tom Wujec의 *Draw Toast* 워크샵을 **그림 없이 카드로** 즐길 수 있도록 만든 웹 게임.

made by **WhiteJune**

---

## 🌟 이 게임을 하는 이유

| 목적 | 내용 |
|---|---|
| 💡 **생각 차이 발견** | "토스트 만드는 법" — 같은 말인데 누구는 식빵→토스터→잼만, 누구는 밀밭→공장→마트→집까지. 머릿속이 얼마나 다른지 눈으로 확인합니다. |
| 🧠 **두뇌 워밍업** | 어려운 회의/수업 시작 전, 가벼운 손풀이로 입을 열게 합니다. |
| 🎨 **그림 부담 ZERO** | 마우스로 그리지 않아도, 카드를 골라 배치하기만 하면 끝. |

원안: [Tom Wujec — Draw Toast (TED Talk)](https://www.ted.com/talks/tom_wujec_got_a_wicked_problem_first_tell_me_how_you_make_toast)

## 🎮 게임 진행 (10~15분)

1. **닉네임 입력** + 타이머 설정 (기본 3분).
2. **카드 배치**: 좌측 팔레트에서 카드 클릭 → 캔버스에 등장 → 마우스로 드래그해 위치 잡기.
3. **연결**: 상단 **🔗 연결모드** ON → 카드 A 클릭 → 카드 B 클릭 → 화살표 자동 생성.
4. **편집**: 카드 더블클릭 → 라벨 텍스트 수정 (한글 OK). `Delete` 키로 카드 삭제.
5. **완성**: 상단 **✅ 완성!** 버튼 → 갤러리에 자동 저장.
6. **비교**: 우측 **🖼 갤러리 열기** → **두 작품 비교** → 카드 둘 골라 *"이렇게 다른 생각!"* 확인.
7. **다운로드**: 💾 **PNG 다운로드** (개별 작품) 또는 📦 **전체 PNG로 받기** (갤러리 합본).

## 🃏 카드 라이브러리 (6 카테고리 × 40장)

| 카테고리 | 카드 |
|---|---|
| 🍞 재료 | 식빵·통밀빵·베이글·버터·잼·꿀·치즈·계란·우유·시리얼 |
| 🔥 도구 | 토스터·전자레인지·오븐·후라이팬·가스레인지·칼·접시·포크 |
| 👤 사람 | 나·가족·친구·손·입 |
| 🏪 장소 | 슈퍼마켓·빵집·냉장고·찬장·식탁·부엌 |
| 🌍 공급망 | 밀밭·농부·공장·트럭·돈 |
| ⚡ 추상 | 전기·가스·시간·온도·시작·끝 |

> **공급망 카테고리**는 "밀밭에서 우리집까지" 같은 **스케일형 사고**를 표현하기 위한 핵심 카드 묶음입니다.

---

## 👥 워크샵 진행 팁 (퍼실리테이터용)

1. **3분 그리기**: 사람들이 동시에 만들도록 (브라우저 1대당 1명, 또는 화면 공유).
2. **공유 라운드**: 한 명씩 갤러리에서 자기 작품 발표.
3. **비교 라운드**: **두 작품 비교** 모드로 가장 다른 두 사람의 작품을 나란히. 카드 수 차이, 사용한 카테고리, "어? 너는 농부부터 그렸어?" 같은 발견을 끌어내세요.
4. **회고 질문**:
   - 왜 ○○ 카드를 썼나요? / 왜 안 썼나요?
   - 누가 사람을 그렸고, 누가 안 그렸나요? 그 이유는?
   - 이 차이가 우리 팀의 실제 일에선 어떻게 나타날 것 같나요?

## 🛠 실행 방법

### A. 브라우저로 바로 (GitHub Pages — 추천)
> 배포된 URL은 저장소 상단의 **About** 섹션을 확인하세요.

### B. 로컬에서 실행
폰트 파일을 `fetch()` 하기 때문에 **로컬 HTTP 서버**가 필요합니다 (`file://` 직접 열기는 일부 기능 제한).

```bash
# Python 3
python -m http.server 8000

# 또는 Node
npx http-server -p 8000
```
→ 브라우저에서 http://localhost:8000 접속

🎯 한 줄로 말하면

"같은 말을 해도 머릿속 그림은 다 다르다 — 그걸 눈으로 확인하는 게임"

🍞 왜 하필 "토스트"인가요?

토스트는 누구나 만들어봤어요. 그래서 "토스트 만드는 법 그려보세요" 하면 다들 "쉽지~" 합니다.

그런데 실제로 그려보면 결과가 완전히 다릅니다:

  • A: 식빵 → 토스터 → 잼 (과정에 집중)
  • B: 프라이팬에 버터 두르고 굽기 (방법에 집중)
  • C: 밀밭 → 공장 → 마트 → 우리집 (스케일에 집중)

→ "쉽다고 다 같은 생각인 줄 알았더니, 사실 우리 머릿속은 완전히 다르네?!"를 깨닫게 합니다.

🤝 그래서 어디 쓰나요?

  • 회의 시작 워밍업: 어려운 주제로 바로 들어가기 전, 입을 풀고 생각의 차이를 인정하는 분위기 만들기
  • 팀 빌딩: 같이 일하는 사람의 생각 스타일을 시각으로 파악
  • 기획·디자인 출발점: "다 같은 그림이 아니구나"를 인식하고 사용자 관점을 인정하는 연습

📝 어떻게 하나요? (10~15분)

  1. 닉네임 쓰고 시작 누르기 (3분 타이머)
  2. 왼쪽 카드 팔레트에서 카드 클릭 → 캔버스에 추가됨
  3. 카드를 마우스로 끌어서 위치 조정
  4. 위쪽 🔗 연결모드 켜고 → 카드 A 클릭 → 카드 B 클릭 → 화살표 자동 연결
  5. 카드를 더블클릭하면 라벨 글자를 바꿀 수 있어요
  6. ✅ 완성! 누르면 갤러리에 저장 + 내 생각 분석 결과 표시
  7. 🖼 갤러리에서 다른 사람과 두 작품 나란히 비교하기

💡 무엇을 느껴야 하나요?

  • 🤯 "어? 너는 이렇게 생각해?" — 같은 토스트인데 나와 다른 부분 발견
  • 🧠 "내 머릿속이 어디까지 펼쳐져 있지?" — 식빵만 본 사람 vs 밀밭부터 본 사람
  • 👤 "누가 사람을 그렸고, 누가 안 그렸지?" — 사용자/사람 중심으로 보는지, 기계·과정만 보는지
  • 🔗 "나는 순서를 중요시했나, 흩어놓았나?" — 화살표를 많이 썼는지 / 카드만 늘어놓았는지
